# Pictionary Frontend

A real-time drawing & guessing game client built with React, Vite & Tailwind CSS.

## Prerequisites

- Node.js v20.x (tested on 20.5.1)
- npm v9.x or yarn v1.x

## Installation

1. **Clone & enter**
   ```bash
   git clone https://github.com/your-org/pictionary.git
   cd pictionary/frontend
   ```
